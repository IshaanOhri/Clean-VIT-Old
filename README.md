<code>
  <h1 align="center">Clean VIT</h1>
  <h3 align="center">The Daily Waste Management System</h3>
</code>

<p align="center">
  An application for students to ensure proper collection of garbage in the campus. The application provides an interface where users can upload complaints and can track its realtime status. The admin gets notified of every complaint with its location, date and time of upload and the site image. The admin also gets the verified user contact details
</p>
